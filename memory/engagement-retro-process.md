Every engagement ends with a 30-minute retro. The reusable lessons — gotchas, estimates that were wrong, assets worth keeping — get written into the company brain by whichever assistant the consultant already uses (Claude, Copilot, ChatGPT, Vally), tagged with the practice area. Retro template at <path>. Owner: Anika Reddy (Strategy & Transformation). This is what keeps the brain current instead of letting knowledge die in a project channel.
